# SpendSense – Expense Tracker App

SpendSense is a mobile application built using Flutter and Firebase that helps users track daily income and expenses, manage budgets, and analyze spending habits.

## Features
- Add and manage income & expenses
- Category-based tracking
- Monthly budget planning
- Data visualization using charts
- Firebase authentication
- Local storage using Hive (offline support)

## Tech Stack
- Flutter (Dart)
- Firebase Realtime Database & Authentication
- Hive (Local Storage)
- Provider (State Management)

## Project Overview
This project was developed as part of my learning to understand mobile app development, backend integration, and data handling.
